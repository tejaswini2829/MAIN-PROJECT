Introduction to python

Python is a high-level, interpreted programming language known for its simplicity and readability.
It's versatile, supporting both procedural and object-oriented programming paradigms.
Python's dynamic typing and automatic memory management make it beginner-friendly.
Its extensive standard library provides modules and functions for various tasks.
Python's indentation-based syntax enhances code readability and reduces the need for braces or semicolons.

Syntax

Python's syntax is designed for readability and simplicity, using indentation to define code blocks rather than braces or keywords.
This enforces clean and consistent code formatting. Variables are dynamically typed, allowing for flexible assignment without explicit declaration of data types.
Control structures like if-else statements, loops (for and while), and function definitions use clear, straightforward keywords.
Functions and classes are defined using the def and class keywords, respectively, with parameters and methods specified within indented blocks.
This design promotes an intuitive and easy-to-follow coding style.

Comments

In Python, comments are used to explain code and are ignored during execution.
Single-line comments start with #, while multi-line comments are enclosed in triple quotes ''' or """.

Datatypes

Python supports various data types including integers (int), floating-point numbers (float), strings (str), lists (list), tuples (tuple), dictionaries (dict), and sets (set).
These data types allow for diverse and flexible data manipulation and storage.

Variables



In Python, variables are used to store data values and do not require explicit declaration of their data types.
They are dynamically typed, meaning the type is inferred from the value assigned.
Variable names must start with a letter or an underscore, followed by letters, digits, or underscores.
For example:
x = 10 # Integer
name = "Alice" # String
pi = 3.14 # Float
is_valid = True # Boolean

Strings



In Python, strings are sequences of characters enclosed in single quotes ('), double quotes ("), or triple quotes (''' or """) for multi-line strings.
They are immutable, meaning once created, their content cannot be changed. Strings support various operations and methods:
Creating String:
single_quote_str = 'Hello'
double_quote_str = "World"
multi_line_str = '''This is a
multi-line string'''
Concatenation:
greeting = single_quote_str + ' ' + double_quote_str # 'Hello World'
Repetition:
repeated_str = "Hi! " * 3 # 'Hi! Hi! Hi! '
Slicing:
sliced_str = greeting[0:5] # 'Hello'

String Methods:
upper_str = greeting.upper() # 'HELLO WORLD'
lower_str = greeting.lower() # 'hello world'
split_str = greeting.split() # ['Hello', 'World']

Strings can also be formatted using f-strings for more readable and efficient string interpolation:
name = "Alice"
age = 30
formatted_str = f"{name} is {age} years old" # 'A

list/set/tuple

In Python, lists, tuples, and sets are used to store collections of items.
Lists are mutable sequences enclosed in square brackets [], tuples are immutable sequences enclosed in parentheses (), and sets are unordered collections of unique items enclosed in curly braces {}.

Booleans

In Python, booleans are a data type that can have one of two values: True or False.
They are commonly used in conditional statements and expressions to control the flow of a program.

If Else

In Python, if and else statements are used for conditional execution of code blocks:
x = 10
if x > 5:

print("x is greater than 5")
else:
print("x is 5 or less")

You can also use elif for multiple conditions:
x = 10
if x > 15:
print("x is greater than 15")
elif x > 5:
print("x is greater than 5 but less than or equal to 15")
else:
print("x is 5 or less")

Switch

Python does not have a built-in switch statement like some other programming languages.
However, you can achieve similar functionality using dictionaries and functions:
def case1():
print("Case 1")
def case2():
print("Case 2")
switch = {
1: case1,
2: case2
}
switch.get(1, lambda: "default")()

While loop

In Python, the while loop is used to execute a block of code repeatedly as long as a specified condition is true:
                    x = 0
                    while x < 5:
                        print(x)
                        x += 1
                This loop will print numbers from 0 to 4, incrementing x in each iteration until x becomes greater than or equal to 5.

For loop

 In Python, the for loop is used to iterate over a sequence (such as a list, tuple, string, or range) or any iterable object:
                    fruits = ["apple", "banana", "cherry"]
                    for fruit in fruits:
                        print(fruit)
                    This loop will iterate over each item in the fruits list and print them one by one.

Arrays

In Python, arrays are not built-in data types like in some other programming languages. Instead, lists are commonly used to store collections of items, including homogeneous or heterogeneous data
                    numbers = [1, 2, 3, 4, 5]
                    names = ["Alice", "Bob", "Charlie"]
                    mixed_data = [1, "hello", True]
                    # Accessing elements
                    print(numbers[0])  # Output: 1
                    print(names[1])    # Output: Bob
                    # Modifying elements
                    numbers[2] = 10
                    print(numbers)     # Output: [1, 2, 10, 4, 5]
                    # Length of the list
                    print(len(numbers))  # Output: 5
                    Python's list type is dynamic and flexible, allowing for easy manipulation and storage of data.

Dictionaries

In Python, dictionaries are used to store key-value pairs. They are defined using curly braces {} and consist of comma-separated key-value pairs, where each key is associated with a value: person = { "name": "Alice", "age": 30, "city": "New York" } # Accessing values print(person["name"]) # Output: Alice print(person["age"]) # Output: 30 # Modifying values person["age"] = 31 print(person) # Output: {'name': 'Alice', 'age': 31, 'city': 'New York'} # Adding new key-value pairs person["occupation"] = "Engineer" print(person) # Output: {'name': 'Alice', 'age': 31, 'city': 'New York', 'occupation': 'Engineer'} # Length of the dictionary print(len(person)) # Output: 4 Dictionaries provide a flexible way to store and retrieve data based on unique keys, allowing for efficient lookup and manipulation of data.
