Introduction to Java

JavaScript Basics: JavaScript is a versatile, high-level, interpreted programming language primarily used for enhancing web pages by enabling interactive features.

Syntax and Variables: It uses a C-like syntax and supports variables defined using var, let, or const.

Data Types: JavaScript has dynamic typing, with basic data types including numbers, strings, booleans, objects, arrays, and null/undefined.

Functions: Functions are first-class objects, allowing them to be stored in variables, passed as arguments, and returned from other functions.

DOM Manipulation: JavaScript can manipulate the Document Object Model (DOM) to dynamically change HTML and CSS.

Events: It handles events such as clicks, form submissions, and page load to create interactive web experiences.

Syntax

Variables: Declared using var, let, or const to store data values. let and const are block-scoped, whereas var is function-scoped.

Data Types: JavaScript has dynamic typing and supports various data types including numbers, strings, booleans, objects, arrays, null, and undefined.

Operators: JavaScript uses arithmetic operators (+, -, *, /), assignment operators (=, +=, -=, etc.), comparison operators (==, ===, !=, !==, etc.), and logical operators (&&, ||, !).

Functions: Functions are reusable blocks of code defined using the function keyword or as arrow functions. They can take parameters and return values.

Conditional Statements: JavaScript uses if, else if, and else to execute code based on conditions. The switch statement is used for multiple potential values of a variable.

Loops: Iterative statements like for, while, and do...while loops are used to execute code repeatedly. for...of is used for iterable objects, and for...in is used for object properties.

Objects: Collections of key-value pairs. Properties can be accessed and manipulated using dot notation or bracket notation.

Arrays: Ordered collections of values, which can be accessed by their index. Arrays support various methods for adding, removing, and manipulating their elements.

Comments: Single-line comments start with //, and multi-line comments are enclosed in /* ... */.

ES6 Features: Modern JavaScript includes enhancements like template literals for string interpolation, destructuring assignment for unpacking values from arrays or objects, and module import/export for organizing code.


Comments

Single-line comments: Indicated by //, everything following // on that line is considered a comment. Single-line comments are useful for brief notes or explanations.

Multi-line comments: Enclosed between /* and */, can span multiple lines. Multi-line comments are ideal for longer explanations, detailed notes, or temporarily disabling blocks of code.

Using comments effectively helps in understanding and maintaining code, facilitating better collaboration among developers.

Variables

In JavaScript, variables are containers for storing data values and are declared using var, let, or const.

var: Function-scoped and can be redeclared; its usage is generally discouraged in modern JavaScript.

let: Block-scoped, introduced in ES6, and prevents redeclaration within the same scope.

const: Block-scoped and used for constants; once assigned, its value cannot be changed.

Variables can store different data types, including numbers, strings, booleans, objects, and arrays.

Operators

In JavaScript, operators are symbols used to perform operations on variables and values. They can be categorized into several types:

Arithmetic Operators: Perform basic mathematical operations.

Examples: + (addition), - (subtraction), * (multiplication), / (division), % (modulus).

Assignment Operators: Assign values to variables.

Examples: = (assign), += (add and assign), -= (subtract and assign), *= (multiply and assign), /= (divide and assign).

Comparison Operators: Compare two values and return a boolean.

Examples: == (equal to), === (strict equal to), != (not equal to), !== (strict not equal to), > (greater than), < (less than), >= (greater than or equal to), <= (less than or equal to).

Logical Operators: Perform logical operations and return a boolean.

Math

In JavaScript, the Math object provides a collection of properties and methods for performing mathematical operations. Here are the key aspects of the Math object:

Constants: The Math object includes several mathematical constants.

Examples: Math.PI (π, approximately 3.14159), Math.E (Euler's number, approximately 2.718).

Basic Methods: Methods for basic arithmetic operations and other common mathematical functions.

Examples:

Math.abs(-5); // Absolute value: 5

Math.round(4.7); // Rounds to the nearest integer: 5

Math.ceil(4.2); // Rounds up to the nearest integer: 5

Math.floor(4.7); // Rounds down to the nearest integer: 4

Math.max(1, 2, 3); // Returns the largest number: 3

Math.min(1, 2, 3); // Returns the smallest number: 1

Trigonometric Methods: Methods for trigonometric calculations.

Example:

Math.random(); // Example output: 0.123456789

These features make the Math object versatile for performing a wide range of mathematical computations in JavaScript.


Strings

In JavaScript, strings are used to represent text and are a fundamental data type. Here are key points about strings:

Declaration: Strings can be declared using single quotes ('), double quotes ("), or backticks (`) for template literals

let str1 = 'Hello';

let str2 = "World";

let str3 = `Hello, ${str2}!`; // Template literal

Concatenation: Strings can be concatenated using the + operator or template literals.

Strings have various built-in methods for manipulation, such as toUpperCase(), toLowerCase(), substring(), indexOf(), split(), and replace().




let upperStr = str1.toUpperCase(); // "HELLO"

let subStr = str2.substring(0, 3); // "Wor"

let index = str2.indexOf('o'); // 1

let parts = str3.split(','); // ["Hello", " World!"]

let newStr = str2.replace('World', 'JavaScript'); // "Hello, JavaScript!"

These features make strings powerful and flexible for handling text in JavaScript.

Booleans

Booleans in JavaScript represent true or false values. They are used in logical and comparison operations, as well as conditional statements. Booleans can be assigned directly with true or false. Logical operators like && (AND) and || (OR) combine boolean values. Comparison operators like ==, ===, !=, !==, >, <, >=, and <= compare values and return a boolean. Booleans are crucial for controlling program flow with conditional statements like if, else if, and else. JavaScript also has truthy and falsy values, where certain values are considered true or false in a boolean context. Understanding booleans is essential for making decisions

If Else

In JavaScript, if statements are used for conditional execution of code. An if statement evaluates a condition and executes a block of code if the condition is true. Optionally, an else statement can be used to execute a different block of code if the condition is false. Multiple conditions can be tested using else if statements.

Example:

let num = 10;

if (num > 10) {

console.log("Number is greater than 10.");

} else if (num === 10) {

console.log("Number is equal to 10.");

} else {

console.log("Number is less than 10."); }



Understanding if and else statements is fundamental for creating dynamic and responsive JavaScript programs.

Switch

The switch statement in JavaScript allows for conditional execution of code based on different cases. It begins with the keyword switch followed by an expression, and then contains case clauses with associated code blocks. An optional default case handles situations where none of the cases match the expression. Each case block typically ends with a break statement to prevent fall-through. Here's a brief overview: br>
Syntax:

switch (expression) {

case value1:

//
Code to execute if expression equals value1

break;

case value2:

// Code to execute if expression equals value2

break;

default:

// Code to execute if no case matches expression }

While loop

A while loop in JavaScript executes a block of code as long as a specified condition is true. Here's a concise overview in 10 lines:

Syntax:

while (condition) {

// Code block to execute while condition is true

}

Example:

let count = 0;

while (count < 5) {

console.log("Count is: " + count);

count++; }

For loop

Syntax:

for (initialization; condition; increment/decrement) {

// Code block to execute while condition is true

} Example:

for (let i = 0; i < 5; i++) {

console.log("Index is: " + i);

}

In a for loop, the initialization statement is executed only once at the beginning. Then, the condition is evaluated. If true, the loop executes the code block, followed by the increment/decrement statement. This process repeats until the condition becomes false.
